# 🐾 HealthyPet

O **HealthyPet** é um sistema em desenvolvimento para facilitar o gerenciamento e a organização das informações dos pets em um único lugar.

O objetivo do projeto é ajudar tutores a acompanharem a saúde, histórico e cuidados dos seus animais de forma simples, prática e acessível.

---

## 📌 Funcionalidades

- 📋 Cadastro de pets
- 💉 Controle de vacinas
- 🩺 Histórico médico
- ⏰ Alertas para próximas vacinas
- 🎂 Lembrete de aniversário do pet
- 📝 Registro de informações importantes
- 🔔 Sistema de notificações e lembretes
- 👤 Cadastro de usuários/tutores

---

## 🛠️ Tecnologias em Utilização

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

---

## 🚀 Objetivo do Projeto

O HealthyPet busca centralizar todas as informações importantes dos pets em uma única plataforma, permitindo maior organização, acompanhamento da saúde animal e praticidade para os tutores.
